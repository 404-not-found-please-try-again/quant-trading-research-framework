## 🔴 必须完成（实盘前必需）

### 1. 模拟盘测试（Paper Trading）

**优先级**: 🔴 最高  
**时间**: 至少 1-2 个月

#### 测试内容：

- [ ] 实时数据获取和处理
- [ ] 信号生成的实时性
- [ ] 订单执行逻辑
- [ ] 异常情况处理（数据缺失、延迟、API 故障）
- [ ] 状态持久化（重启后恢复持仓）
- [ ] 性能监控和日志记录

#### 评估标准：

- 模拟盘收益与回测收益偏差 < 20%
- 系统稳定性 > 99%
- 数据延迟 < 5 秒
- 无重大逻辑错误

## ⚠️ 风险提示

1. **回测不等于实盘**: 回测收益不能保证实盘收益
2. **市场环境变化**: 模型可能在不同市场环境下表现不同
3. **技术风险**: API 故障、系统崩溃等
4. **流动性风险**: 大额交易可能影响价格
5. **数据质量风险**: 数据延迟、错误等
